# BARNA Website
Official Website for BARNA Group in Bangalore. Access it [here](https://barna.vercel.app)

## Setup
1. Make sure you are using the right `node` version using [nvm](https://github.com/nvm-sh/nvm)
```sh
nvm use
```
2. Install all npm depenencies
```sh
npm i
```
